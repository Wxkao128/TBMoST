### 計算物體的質量

假設我們有一個物體，根據愛因斯坦的質能等價公式：
    
    .. math::
        E = mc^2

其中，`E` 是能量，`m` 是質量，`c` 是光速。

### 程式碼範例

以下是用 Python 實現該公式的程式碼：

.. code-block:: python

    def energy(mass):
        c = 3 * 10**8  # 光速，單位：m/s
        return mass * c**2

我們可以使用這個程式碼計算不同質量物體的能量。

### 計算結果

假設物體的質量為 1 公斤：

    .. math::
        E = 1 \times (3 \times 10^8)^2 = 9 \times 10^{16} \, \text{J}

這將是一個巨大的能量。
